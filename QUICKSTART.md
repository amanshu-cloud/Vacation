# ⚡ Quick Start (30 seconds)

## 1. Get API Key
Visit https://console.anthropic.com, create an account, go to API Keys, and copy your key.

## 2. Configure
Edit `.env` file and replace the placeholder:
```
ANTHROPIC_API_KEY=sk-ant-your-actual-key-here
```

## 3. Run
```bash
npm run demo
```

Watch Claude analyze a sample project in real-time! 🎉

## 4. Try Interactive Mode
```bash
npm run workflow
```

Paste your project description and ask Claude to break it down.
